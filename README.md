# 🎬 MovixHub

MovixHub is a modern movie discovery web application that allows users to explore, search, and interact with movies in a clean and responsive interface.

---

## 🚀 Live Demo
🔗 https://movixhub-rho.vercel.app/ 

---

## 📌 Features

- Browse popular movies  
- Explore upcoming releases  
- Real-time movie search with suggestions  
- Watch trailers directly inside the app  
- Add and manage favorites (authentication required)  
- Email/password authentication  
- Google sign-in (Firebase)  
- Fully responsive design (mobile + desktop)  
- Smooth UI with glass-inspired design  

---

## 🛠 Tech Stack

- **Frontend:** React (Vite)  
- **Styling:** Tailwind CSS  
- **Authentication:** Firebase Auth  
- **API:** TMDB (The Movie Database)  
- **Deployment:** Vercel
